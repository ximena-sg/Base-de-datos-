# Base de datos

Desarrollo:
1. Buscar un conjunto de datos de interés. Se facilitan algunos sitios:
● https://www.datos.gob.ar/dataset
● https://data.buenosaires.gob.ar/dataset/
● https://www.tableau.com/es-es/learn/articles/free-public-data-sets
● https://www.kaggle.com/datasets

2. Una vez elegido, se deben explicar el origen de los datos, ubicación geográfica, etc, detallando sus variables y aplicar los pasos del análisis
exploratorio comentando brevemente las observaciones.

3. Una vez que los datos están listos, se deben crear los modelos (según corresponda) de regresión lineal múltiple o regresión logística, con las
correspondientes observaciones. Y considerar aplicar validaciones, optimizaciones y regularizaciones, comentando brevemente las observaciones.
